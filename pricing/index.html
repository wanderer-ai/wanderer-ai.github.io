Pricing

For this project I will pursuing an open core strategy. That means that the core and most of the packages will be available under AGPLv3. That means, that you can build and run basic projects for free. The rest of the packages will be available under a commercial license. So you have to purchase a license if you plan to host a chatbot using a commercial package on a public domain.

The pricing model is not very clear right now. But I want to keep it simple. So you have to purchase a license code for each domain for each commercial package you are using.
