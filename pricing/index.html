Pricing

This software is open source but it is not completely free. That means that you have to purchase a license if you plan to run the botbuilder or single chatbots on your own website or inside your apps. Use the botbuilder and the chatbots for free as long as you use them with the Wanderer.ai domain.

The pricing model is not very clear right now. But I want to keep it simple. So you have to purchase a license code for each domain or App on which one or more bots are running.
